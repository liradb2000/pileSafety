# 프로그램 상세

## 간단 안내

본 프로그램은 그래픽 창(좌측)과 값 입력 창(우측)으로 나눠져있습니다.

![create_project](/img/program/programdetail.gif)

1. 우측 값 입력 창에서 **검토** 클릭
1. 변수 (값) 입력
1. **Apply** 클릭

## 변수 및 버튼 상세 안내

### 오염반경 변수

| 변수명 | 설명          |
| ------ | ------------- |
| a      | 터파기의 폭   |
| b      | 터파기의 높이 |

### 대수층 두께 검토 변수

| 변수명   | 설명          |
| -------- | ------------- |
| 굴착깊이 | 터파기의 폭   |
| 근입깊이 | 터파기의 높이 |

### 토질영향반경 변수

| 변수명   | 설명        |
| -------- | ----------- |
| 토질입경 | 터파기의 폭 |

### 투수계수 변수

| 변수명 | 설명          |
| ------ | ------------- |
| c 계수 | 터파기의 폭   |
| t 온도 | 터파기의 높이 |
| D_20   | 터파기의 높이 |

### 결과 파일 변수

| 변수명      | 설명        |
| ----------- | ----------- |
| 데이터 간격 | 터파기의 폭 |
